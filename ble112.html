UoA Bluegiga BLE112

PCB DESIGN • ALTIUM • EMBEDDED SYSTEMS

This was done as part of a voluntary internship at the University of Auckland from November 2020 to February 2021.

The Bluegiga BLE112 is a Bluetooth Low-Energy module that was primarily used for communication between a remote and a steering controller. The PCB was designed based on information from the datasheet and for a small form factor to be modular. The software for the BLE112 on the other hand was implemented using BGScript, a simple BASIC-style programming language for developing applications on BlueGiga BLE modules.
